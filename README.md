**vimrc.bundles**
1. git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim
2. 修改 .vimrc
3. 打开vim在底行模式下输入 BundleInstall
4. 如果还需要其他插件，可以使用BundleSearch 进行在线查找插件
